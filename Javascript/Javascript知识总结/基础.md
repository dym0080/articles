
# 对象

`object`类型指的是一种复合值，你可以在它上面设定属性（带名称的位置），每个属性持有各自的任意类型的值。它也许是`JavaScript`中最有用的类型之一。
```js
var obj = {
	a: "hello world",
	b: 42,
	c: true
};

obj.a;		// "hello world"
obj.b;		// 42
obj.c;		// true

obj["a"];	// "hello world"
obj["b"];	// 42
obj["c"];	// true
```
属性既可以使用 点号标记法（例如，`obj.a`） 访问，也可以使用 方括号标记法（例如，`obj["a"]`） 访问。点号标记法更短而且一般来说更易于阅读，因此在
可能的情况下它都是首选。

有时对象的属性可能是动态的，可能被复制到一个变量中，这时候方括号标记法会很有用了，例如：

```js
var obj = {
	a: "hello world",
	b: 42
};

var b = "a";

obj[b];			// "hello world"
obj["b"];		// 42
obj.b;		  //42
```
